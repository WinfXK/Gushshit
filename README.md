## Gushshit 是什么?

一个[NukkitX](https://github.com/NukkitX/Nukkit)多功能自定义GUi玩家公告插件

## Gushshit 有哪些功能？
> - 拥有高度自由的可配置文件
> - 多种公告方式
> 	- TIp
> 	- Popup
> 	- Message
> 	- Title
> - 傻瓜式的UI操作
> - 支持多种计费
> - 支持多种颜色设定
> - 支持Title显示时间设定
> - 往下看！您将会收获更多！
